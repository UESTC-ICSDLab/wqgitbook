# LAB1: "施法"让 CPU 动起来
<!-- -->
> #### important::README
> 1. 在本实验 LAB1 中, 你将动手学习 Keil 开发环境的搭建, 基础汇编程序的编写, 编译和仿真调试方法, 并通过几个简单程序理解函数调用的 ATPCS 规则.
